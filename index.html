NailAI — Fine-Grained Medical Vision System

A fine-grained medical image classification system built to balance predictive quality, long-tail sensitivity, explainability, and real deployment constraints.

93%
Accuracy
30%

5×

Problem

The system had to classify 22 visually similar nail disease categories while handling class imbalance and keeping inference fast enough for cloud serving.

What I Built

I trained and evaluated ResNet-based classifiers, improved minority-class behavior with hierarchical classification and oversampling, added Grad-CAM explainability, and prepared the service for deployment with Docker and Cloud Run.

Technical Decisions
  • Compared ResNet18 vs ResNet50 using both accuracy and latency.
  • Selected the lighter architecture because it preserved quality and reduced serving cost.
  • Used class balancing and hierarchical logic to improve clinically sensitive classes.
  • Included explainability for debugging and more responsible interpretation.
Outcome
  • Reached 93% accuracy on a 22-class fine-grained problem.
  • Improved minority-class F1 by 30%.
  • Reduced inference time by 5× compared with the heavier baseline.
  • Produced a deployment-ready inference workflow rather than a notebook-only result.

Vision-Based Imitation Learning for Go1 Navigation

A VLM-driven robot navigation workflow for Unitree Go1, centered on front-camera scene understanding, structured action decisions, ROS integration, and safety-aware deployment constraints.

VLM
Navigation Policy
ROS

Problem

Build a robot navigation policy from front-camera perception that can reason about people and motion in the scene, while remaining safe enough to use in real robot workflows.

What I Built

I built a VLM-based social navigation workflow for Go1 that turns front-camera observations into structured navigation decisions, connects those decisions to a ROS-based robot stack, and evaluates what part of the policy is safe to expose on a real robot.

Technical Decisions
  • Used pretrained VLMs as the high-level navigation decider for front-camera social navigation.
  • Separated semantic navigation decisions from low-level control through a safety-preserving controller layer.
  • Focused on what survives real robot deployment, including latency, stop behavior, and lateral-motion safety.
  • Evaluated the policy as a robotics system, not just as an offline benchmark.
Outcome
  • Produced a VLM navigation workflow that could be audited offline and connected to real robot control.
  • Clarified the gap between benchmark policy performance and safe realtime deployment.
  • Strengthened my robotics, autonomy, and deployment-focused systems experience.
